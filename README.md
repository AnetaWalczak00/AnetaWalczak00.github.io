# Dokumentacja projektu
BALANCR - system do zarządzania domowym budżetem.

## Link do dokumentacji:
 **https://AnetaWalczak00.github.io**

---

## O projekcie

BALANCR to aplikacja internetowa umożliwiająca użytkownikom planowanie, monitorowanie i analizowanie domowego budżetu w nowoczesny i przystępny sposób.  
Projekt realizowany jest w ramach przedmiotu **PPSI II oraz WDZPD**.

---

## Technologie

- **Frontend:** Next.js, React  
- **Backend:** C#, .NET  
- **Baza danych:** PostgreSQL
- **Projekt graficzny:** Figma  
- **Kontrola wersji:** Git, GitHub  
- **Zarządzanie projektem:** Jira  
- **Hosting dokumentacji:** GitHub Pages
