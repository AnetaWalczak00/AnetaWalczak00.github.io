O projekcie

Projekt systemu internetowego na zaliczenie przedmiotu PPSI II i WDZPD – System zarządzania domowym budżetem

Opis projektu

BALANCR to aplikacja internetowa umożliwiająca użytkownikom planowanie, monitorowanie i analizowanie domowego budżetu w przystępny i nowoczesny sposób.

Technologie

  • Frontend: Next.js + React
  • Backend: C# + .NET
  • Baza danych: do ustalenia
  • Projekt graficzny: Figma
  • Kontrola wersji: Git + GitHub
  • Zarządzanie projektem: Jira
  • Hosting strony - dokumentacja: GitHub Pages
